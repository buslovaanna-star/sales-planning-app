# 📊 Планування продажів магазинів

Streamlit-застосунок для аналізу та планування продажів мережі магазинів.

## Функціонал

- **Завантаження даних** — Excel-файл з фактичними продажами по магазинах
- **Авто-план** — розрахунок плану з урахуванням тренду, сезонності та віку магазину
- **Редагування** — глобальний приріст +X% або правка кожного магазину вручну
- **Аналітика** — виконання плану, рейтинг магазинів, графіки
- **Рекомендації** — автоматичне виявлення магазинів із систематичним відхиленням

## Запуск локально

```bash
pip install -r requirements.txt
streamlit run sales_planning_app.py
```

## Формат вхідного файлу

Excel-файл повинен містити лист **"ДАНІ ВНОСИТИ"** такої структури:
- Рядок 1: назви магазинів (починаючи з колонки C)
- Рядки 4+: дати (колонка A) та факт продажів по кожному магазину

Опціонально: лист **"Коеф. Сезонності"** з місячними коефіцієнтами сезонності.

## Розгортання на Streamlit Cloud

1. Завантажте репозиторій на GitHub
2. Зайдіть на [share.streamlit.io](https://share.streamlit.io)
3. Підключіть репозиторій → виберіть `sales_planning_app.py`
4. Натисніть Deploy
